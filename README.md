# 动态路由后端

## 项目描述

根据主机telnet连接网络设备的技术以及Tcl或python等技术，实现基于Web的网络拓扑搭建及校验的自动化。

> telnet连接网络设备的技术参考附件文件
>
> telnet访问交换机：https://wenku.baidu.com/view/b2410527a31614791711cc7931b765ce05087ae5.html
